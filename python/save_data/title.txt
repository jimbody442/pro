파일에 저장하는 것도 데이터 전송.
데이터를 전송하기 위해서는 전송을 위한 어떤 형태로 변형, 파일에 저장하거나 네트워크로 전송하기 위해서는
바이트로 이루어진 리스트 같은 형태의 데이터로 변형.
-이렇게 메모리에 있는 데이터를 바이트 스트림으로 변경시키는 것을 "직렬화"
-앞으로 소개할 모듈은 모두 데이터 직렬화하고 역직렬화 해주는 모듈,

              ----pickle---------json----------marshal------
-객체 중복처리--     O             X               X
-저장 포맷--     바이너리        텍스트          바이너리
-사용자정의--        O             X               X
-버전별 호환--       O             O               X 
-주요 사용처--  파이썬 객체저장--통신 위함 ---- 컴파일된 파이썬 코드 저장용용
